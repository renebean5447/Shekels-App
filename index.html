🗺️ Welcome to Shekels — Feature Guide

💰 Add Entry Card ▾
  • Enter an Amount, Hours, and Minutes worked.
  • Select or add a Platform (click + to type a new one, press Enter to confirm).
  • Your selected platform stays active after each entry — no need to re-select it!
  • Use the 🗑️ button to delete an unwanted platform from the list.
  • Click an entry in the Entry Log to edit the amount, hours, minutes, and platform.
  • Choose the Currency of the entry — it auto-converts to your default.
  • Click Add or press Enter in any field to log the entry.
📋 CSV Import — Format Guide ▾

Go to Settings → Import CSV to upload a .csv file. Shekels will read as many columns as it can match.

Recognized column names (case-insensitive):

  • amount or earnings or $ — Required. The dollar amount earned.
  • date — Date of the entry. Accepts YYYY-MM-DD or MM/DD/YYYY. Defaults to today if missing.
  • hours or hour — Whole hours worked.
  • minutes or minute or mins — Minutes worked (0–59).
  • platform or source or client — Platform or client name. Auto-added to your platform list.
  • currency — 3-letter currency code (e.g. USD, EUR). Amounts will be converted to your default currency.
  • notes or note or description — Stored with the entry log record.

Example CSV:

amount,date,hours,minutes,platform,currency,notes 45.00,2025-06-10,2,30,Upwork,USD,Logo design 120.50,2025-06-11,4,0,Fiverr,USD,Website revamp 30.00,2025-06-12,1,15,Direct Client,USD,Quick consultation 200.00,06/13/2025,6,0,Toptal,EUR,Sprint work

Tips:

  • Save your spreadsheet as .csv (Comma delimited) from Excel or Google Sheets.
  • Column order does not matter — Shekels reads by header name.
  • Rows with a missing or invalid amount will be skipped automatically.
  • Imported entries appear in the Entry Log and update all dashboard cards.
👤 Profiles ▾
  • Use File → Save As New Profile to save your current data and layout as a named profile.
  • Use File → New Fresh Profile to create a brand-new blank profile under a new name.
  • Switch between profiles using the Settings → Profile dropdown. Each profile stores its own entries, layout, and all customization settings independently.
  • The active profile is shown in the toolbar (e.g. 👤 Default).
  • Profiles auto-save every few minutes. You can also hit File → Save at any time.
🏆 Goal Card ▾
  • Set an earnings target with a custom duration — including sub-day hour-based goals!
  • The progress bar fills as you earn. Countdown shows time remaining.
  • When you hit 100%, fireworks celebrate your achievement (can be disabled in Settings)!
  • After reaching the goal you'll be prompted to set a new one or keep going above 100%.
  • Use 🎯 New Goal to set a fresh goal at any time (resets all progress).
  • Use 🔄 Reset Goal to clear the goal entirely.
📅 Today Card ▾
  • Shows today's Total Earnings, Avg per Hour, and Time Worked.
  • Resets automatically at midnight every day.
📈 Chart Card ▾
  • A line graph of your earnings by day with date labels.
  • Use ↑+ / ↑− to zoom in and out on the Y axis (value scale).
  • Use ↺ Reset to restore the default view.
  • You can also scroll to zoom and drag to pan on desktop.
⚙️ Settings & Customization ▾
  • Themes: 11 built-in color themes.
  • Currency: 10 currencies with live exchange rates.
  • Custom colors: Background, box background, text, and card border (hue slider).
  • Fonts: 13 font choices including Roboto, Poppins, Montserrat, Lato, Open Sans, and Raleway.
  • Font size fully customizable.
  • Background image: Upload any image as your workspace background.
  • Card border width: 0–20px slider for thick or thin borders.
  • Animations: Toggle falling money and goal fireworks independently.
  • Autosave: Auto-saves every 5–120 minutes. Motivational quote refreshes on each autosave.
🃏 Extra Cards ▾
  • Stats Summary — see all-time totals, average session, and best day at a glance.
  • Notes — a free-form scratchpad saved with your profile.
  • Streak Tracker — tracks how many consecutive days you've logged earnings.
  • Hourly Rate Tracker — shows your effective hourly rate across different periods.
  • Find all of these in the Add/Restore Cards menu in the toolbar.
